飞秒激光加工包层波导的研究计划与分析报告

执行摘要

本文所讨论的“包层波导”，特指飞秒激光直写凹陷包层波导，亦即文献中常说的 depressed-cladding waveguide 或 Type III waveguide。它的核心思想不是把导光芯区直接写成高折射率，而是在芯区外围写入一圈或多圈低折射率改性轨迹；芯区本身通常保持接近原始材料状态，因此在激光晶体、非线性晶体和各向异性晶体中，往往比 Type I/II 结构更容易同时兼顾导光、增益保持、非线性保持和偏振兼容性。现有研究已在 Nd:YAG、LiNbO₃、Pr:YAG、Tm:YAG、蓝宝石、ZBLAN 玻璃以及 LAS 玻璃陶瓷等平台验证了这一路线。
