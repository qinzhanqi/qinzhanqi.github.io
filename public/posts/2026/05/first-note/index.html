2. 行内公式示例

在光学中，数值孔径可以写成 ( NA = n \sin\theta )，其中 ( n ) 是介质折射率，( \theta ) 是最大收光半角。

衍射极限下的横向分辨率可近似表示为：

[ \Delta x \approx 0.61 \frac{\lambda}{NA} ]

其中，(\lambda) 是光波长，(NA) 是物镜数值孔径。

3. 块公式示例

高斯光束的强度分布可以写成：

[ I(r) = I_0 \exp\left(-\frac{2r^2}{w^2}\right) ]

其中：

  • (I_0) 是中心光强；
  • (r) 是径向坐标；
  • (w) 是光束腰半径。

对于 (N) 光子吸收过程，有效吸收分布近似满足：

[ I_N(r) \propto I^N(r) = I_0^N \exp\left(-\frac{2Nr^2}{w^2}\right) ]

因此，非线性吸收可以在一定程度上压缩有效作用区域。
